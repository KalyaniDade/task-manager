# 📌 Team Task Manager

A full-stack web application for managing projects and tasks with role-based access control (Admin/Member). Built using Flask and SQLite.
## 🚀 Features

- User Authentication (Signup/Login)
- Role-Based Access Control (Admin / Member)
- Project Management (Create & Manage Projects)
- Team Management (Add Members to Projects)
- Task Creation and Assignment
- Task Status Tracking (To Do / In Progress / Completed)
- Overdue Task Highlighting
- Dashboard for Admin and Members


## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: SQLite
- ORM: SQLAlchemy

## 👥 User Roles

### 🔹 Admin
- Create projects
- Add team members
- Create and assign tasks
- View all tasks and progress

### 🔹 Member
- View assigned tasks only
- Update task status
- Track personal progress
